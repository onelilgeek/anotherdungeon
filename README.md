# Another Dungeon V2
[Another Dungeon 페이지로 이동](https://hu-lee.github.io/anotherdungeon/)

### 언어 지원
한국어, 日本語, English

### 기능 소개
#### 1. 던전 정보
- 던전 정보와 직업서 드랍 정보 등을 확인 가능합니다.    
- 맵 Download 기능   

#### 2. 직업서 검색
- 캐릭터, 책 이름으로 직업서 드랍 장소 검색   
- 해당되는 던전으로 바로 이동 가능

#### 3. 퍼스널리티 검색
- 선택한 퍼스널리티에 해당하는 캐릭터를 조회
